SAPA - Link in Bio SaaS

A lightweight, fast, and aesthetic Link-in-Bio platform powered by Vanilla JS & Supabase.

The Challenge

Many "Link in Bio" tools are bloated, slow, or restrictive with their free plans. I wanted to create a Mini SaaS solution that is lightweight, performs exceptionally fast on mobile devices, and offers premium features like music embedding and custom backgrounds without relying on heavy frontend frameworks.

The Solution

SAPA (Satu Page Aja) was built using Vanilla JavaScript to ensure maximum performance and zero bloatware. It utilizes Supabase for secure authentication and real-time database management. The platform features a drag-and-drop interface for link management and instant live previews.
