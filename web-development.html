Redefining Student Organization's Digital Presence

Transforming the English Department Student Association (EDSA) website from a basic information page into a dynamic, professional hub for students and events.

The project aimed to deliver a modern user experience prioritizing visual engagement and accessibility. By adopting a mobile-first philosophy and utilizing the robust Bootstrap 5 framework, we created a platform that is not only visually stunning but also technically sound and highly responsive across all devices.

Key Features Implemented

  • Modern Homepage Experience: An immersive Hero section with smooth entry animations, JS-powered animated statistics counters, and a dedicated video highlight section to immediately engage visitors.
  • Interactive Masonry Gallery: A dynamic "Pinterest-style" layout that intelligently arranges portrait and landscape photos. Features a "Focus Mode" modal with a backdrop blur effect for detailed viewing.
  • Fully Responsive & Adaptive: The layout seamlessly adapts to any screen size—from large desktops to smartphones—ensuring a consistent experience without breaking the design.
  • Multimedia & Structure Integration: Specialized sections for Podcasts/Videos using Owl Carousel sliders, and comprehensive pages detailing organizational structure (Divisions like HRD, IT, ECO).
